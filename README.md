# YPAGNSchool2026
Computational Astrophysics and Astronomy Summer School
This 10-day intensive winter school provides a comprehensive introduction to computational astronomy. Participants will progress from Python basics to advanced topics such as spectral analysis, Active Galactic Nuclei (AGNs), and large-scale cosmological simulations. A core focus of this session is the transition from observational data to theoretical modeling. We will explore the IllustrisTNG project, a suite of state-of-the-art magnetohydrodynamic simulations and learn to interface with this data. 
• Duration: 3rd – 13th (+2 days for project showcase) June 2026 (4:00 PM – 6:00 PM IST) 
• Mode: Live Lectures + Hands-on Coding Labs 
• Prerequisites: Basic understanding of physics and mathematics; no prior coding experience required.
